This directory contains the script required to reproduce Figure S2A, showing SparCC values (sparse correlations for compositional data) for pairwise relationships between PECHi-C contact scores in transitioning and primed cells (see Methods in Nocente et al.).

To reproduce this plot, run the script "SparCC_correlation_SuppFigure_S2A.R" using the following input files:
- peakMatrix (Focused_peakmatrix_PE_TSS_interaction.tsv), available on OSF.
- spoutT_file (sparcc_n_p_transition_corr.txt), available in this directory.
